# YouthAI Lab — Deliverables

Public deliverables of the YouthAI Lab project.

## What's inside

### `ComprehensiveApproach/` — D2.1
Full programme overview, methodology, and the 6-phase approach. One subfolder per language (EN / FR / IT / CA / NL) with the markdown source and (when available) the published PDF.

### `ExplorationProtocol/`
Exploration sheet protocol + report template used during the workshops. One subfolder per language (EN / FR / IT / CA / NL).

### `Ideation_Sheets/`
Ideation sheets for the workshop "exploration" activities — ready-to-use PDFs covering different AI topics:
- Bio-Inspired Algorithms
- Illusions
- Introductory AI Game · Predictive AI
- Keep Eye on AI
- Looks Right
- The Stupid Invention
